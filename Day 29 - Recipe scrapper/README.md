Quick and Dirty script for getting a recipe from a specific site
https://www.delfi.lv/tasty/receptes

## Prerequisites

- Pushbullet API key
- BeautifulSoup
- PyDotenv

```
pip install -r requirements.txt
```

## Usage

```
python tasty-rip.py linkt.to/recipe
```
